# 银行卡格式化 formatBankNo

## 方法介绍
### 1、同步or异步
> **同步方法**

### 2、使用场景
> 获取用户银行卡号时，使用四位分割展示

## 使用案例
```js
import { formatBankNo } from 'ph-utils'

const bankNo = formatBankNo('661723452876123') // bankNo -> '6617 2345 2876 123'
```

## 参数列表
### 入参

| 序号 | 名称 | 类型 | 是否必须 | 备注 |
| --- | --- | --- | --- | --- |
| 1 | bankNo | string | 是 | 银行卡号 |

### 返回值

| 序号 | 名称 | 类型 | 备注 |
| --- | --- | --- | --- |
| 1 | - | string | 格式化之后的字符串，四位间隔 |
